# lexertoy

A simple lexer in C that reads a file and returns a stream of tokens.
Can be used for a compiler or syntax highlighter, or something completely different!

## Example

Input:
```
// This is a comment
main :: () {
    int x = 100;
    int y = 0xff;
    int z = 0b1001;
    float w = 0.125;

    if (x * y > z + w) {
        println("Hello, World!"); test;
    }

    while (x < 400) {
        x += 4;
    }
}
```
